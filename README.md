# Airbnb Business Website Template

A simple, aesthetic single-page website starter for your Airbnb business.

## Project Structure

- `index.html` - Main page layout and placeholder content sections.
- `assets/css/styles.css` - Site styles, colors, typography, layout, and animations.
- `assets/js/main.js` - Small interaction enhancements (scroll reveal).
- `assets/images/` - Add your photos here.

## How to Use

1. Open `index.html` in your browser.
2. Replace text placeholders with your real Airbnb content.
3. Add images to `assets/images/` and update the image areas in `index.html`.
4. Update links for booking, contact, and social profiles.

## Quick Customization Areas

- Brand name in header: `Grace@C`
- Hero title and subtitle in `index.html`
- Nearby spots cards in the "Popular Spots Nearby" section
- Color palette in CSS variables under `:root`
